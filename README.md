## 🛠 Установка и настройка

### Требования

- Python 3.8 или выше
- Telegram Bot Token (получить у [@BotFather](https://t.me/botfather))
- Сервер с Linux (рекомендуется Ubuntu 20.04+)

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/Krimex1/python-ddos-.git
cd python-ddos-

# Выбор версии
# Для STABLE:
python stresser1.0.py

# Для PRO:
python stresser1.0.1.py

# Для UNLEASHED:
python stresser1.0.2.py
```

### Настройка конфигурации

1. Откройте выбранный файл в редакторе
2. Найдите секцию с настройками бота
3. Замените `YOUR_BOT_TOKEN` на токен вашего бота
4. Настройте параметры под ваше оборудование

```python
# Пример конфигурации
BOT_TOKEN = "your_telegram_bot_token_here"
ADMIN_ID = 123456789  # Ваш Telegram ID
MAX_THREADS = 500     # Максимум потоков
```
